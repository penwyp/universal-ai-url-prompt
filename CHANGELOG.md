# Changelog

All notable changes to this project are documented in this file.

## [0.0.1] - 2026-03-01

### Added

- Platform contract validation pipeline.
- Platform-isolated config files and shared runtime architecture.
- Strict runtime mode with explicit platform selector requirements.
- Playwright E2E contract tests across supported platforms.

### Changed

- Auto-send flow now avoids early form submit before a send control is detected.
- ChatGPT send-button selector coverage expanded.
- Wenxiaoyan delayed-button E2E assertion changed to use actual enable timestamp.
